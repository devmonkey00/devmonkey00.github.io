# Asher Tutanes - Backend Developer Portfolio

A modern, responsive portfolio website showcasing backend development skills, projects, and experience. Built with HTML5 and CSS3, featuring a Monokai-inspired dark theme.

## 🌐 Live Site

Visit the portfolio at: [https://devmonkey00.github.io](https://devmonkey00.github.io)

## ✨ Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations and hover effects
- **Dark Theme**: Monokai-inspired color scheme with dark backgrounds and vibrant accents
- **Sections**:

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/devmonkey00/devmonkey00.github.io.git
   ```

2. Open `index.html` in your web browser to view the portfolio locally.

3. To deploy to GitHub Pages:
   - Push changes to the `main` branch
   - GitHub Pages will automatically deploy the site

## 📝 Customization

To customize this portfolio for your own use:

1. Update personal information in `index.html`:
   - Name and title
   - Experience years
   - Projects and descriptions
   - Skills and technologies
   - Social media links

2. Modify colors in `styles.css`:
   - Edit CSS variables in the `:root` selector
   - Adjust gradients and accent colors as needed

3. Add or remove sections as desired
